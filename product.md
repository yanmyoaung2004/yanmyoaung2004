What you're seeing is a GitHub Profile README. It appears at the top of your GitHub profile page, above your repositories.

For example:

https://github.com/torvalds
https://github.com/sindresorhus
https://github.com/kamranahmedse

all have a profile README.

How to create one

Since your username is:

yanmyoaung2004

you need to create a repository with exactly the same name:

yanmyoaung2004

It must exactly match your GitHub username.

Step 1

Click New Repository

Repository name:

yanmyoaung2004

GitHub will even show a message like:

"You found a secret! yanmyoaung2004 is a special repository that you can use to add a README.md to your GitHub profile."

Step 2
Public
Check Add a README file
Create Repository
Step 3

Edit the README.md.

Whatever you write there becomes your profile.

is it has to be public repo?

Yes. It must be a public repository for your profile README to appear on your public GitHub profile.

The requirements are:

Repository name must exactly match your GitHub username:

yanmyoaung2004
It should contain a README.md file.
The repository must be Public if you want visitors to see it on your public profile.

If you make it Private:

You can still edit the README for yourself.
Public visitors will not see it on your profile.
A few additional tips
